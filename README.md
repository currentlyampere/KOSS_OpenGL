## What did you learn from completing the task
I learned a lot of things from this project. Firstly, ofcourse, how Open GL works, its whole pipeline and how data flows between CPU and GPU. It was kind of overwhelming that we have to put much effort just to put a triangle on a screen. I learned how GPU calculates so much just to put a single pixel on screen. Also, I was fascinated by the use of Linear algebra here. I didn't get the time to implement it but we basically move, rotate, scale the object using matrices (linear transformations). Thanks to 3b1b's Linear algebra series , i already knew how matrices work as linear transformations. Also, blinn-phong lighting was also interesting thing to research about. Specially, how we try to mimic(or fake) real life physics using vectors and stuff.
(Funfact:I have POW, PDS and LANCA in this sem and this project was pretty much intersection of the 3 subjects) 
I wanted to implement it on a sphere to see how it intracts with light.i have a lot projects idea i want to build myself in Open GL, however, again, time was limited.
